HTML Block and Inline Elements

Block level elements starts a new line and its width stretches the whole width available of its parent container. This paragraph right now is a p tag that is block level element.

Inline elements on the other hand only takes up space that is necessary. The span element is a good example of this.

HTML Div Element

The div element is used as a container for other HTML.

HTML Classes

Class atttribute of html element is used to point at in the styling sheet css. Also, Javascript can also use it to access the specific class. An element can have multiple classes inside the class attribute and they are separated by spaces (ex: class="big red").

Different elements can share the same class.

HTML Id

HTML Ids are used to give an element a unique id. They can only be used for one element in the page (have to be unique). It is a pointer for CSS and Javascript as well.
